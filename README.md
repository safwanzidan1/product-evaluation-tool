# أداة تقييم المنتجات

أداة ويب عربية احترافية لتقييم منتجات التجارة الإلكترونية بناءً على نظام نقاط دقيق.

## الميزات

- **واجهة عربية RTL** — تصميم كامل بالعربية
- **Dark Mode / Light Mode** — قابل للتبديل
- **حفظ تلقائي** — localStorage (لا يوجد backend)
- **تصدير واستيراد CSV** — مع ترميز UTF-8 BOM للغة العربية
- **بحث، فلترة، ترتيب** — حسب التصنيف والتقييم والاسم

## الحقول

### المعلومات الأساسية
- اسم المنتج
- رابط المنتج
- سعر بيع المنتج

### معايير التقييم (Checkboxes) — كل ✓ = نقطة واحدة (13 نقطة كحد أقصى)
1. يوفر حل لمشكلة العميل
2. خامات عالية الجودة
3. مناسب لجميع الفئات
4. غير قادر على تخمين السعر
5. غير متوفر في المنافذ
6. موسمي أم لا
7. سهل الشحن والتغليف
8. خفيف الوزن
9. غير قابل للكسر
10. المنتج سهل بيعه
11. القدرة على تحويل المشاهدات لبيع
12. المنتج يتم شراؤه أكثر من مرة
13. المنتج متاح له upsell و cross sell

### التقييم التفصيلي (Scoring) — من 1 إلى 3 (21 نقطة كحد أقصى)
1. الكريتيف
2. حجم المنتج
3. حل المشكلة
4. مناسب للرجال والنساء
5. منتج موسمي
6. Bundle
7. قابل للتصنيع

## معادلة التقييم النهائي

```
التقييم النهائي/10 = (مجموع Checks / 13) × 5 + (مجموع التقييم / 21) × 5
```

## التصنيف

| التقييم | التصنيف |
|---------|----------|
| ≥ 8.5 | 🏆 Winner |
| ≥ 7.0 | ✅ Very Good |
| ≥ 5.0 | 🔵 Good |
| < 5.0 | 🚫 Escape |

## النشر على Netlify

1. اسحب مجلد `product-evaluator` إلى [app.netlify.com/drop](https://app.netlify.com/drop)
2. أو استخدم Netlify CLI:
   ```bash
   npm install -g netlify-cli
   netlify deploy --dir . --prod
   ```

## المتطلبات
- لا يوجد backend
- لا يوجد قاعدة بيانات
- لا يوجد تسجيل دخول
- يعمل على كل متصفح حديث
- البيانات محفوظة في localStorage
